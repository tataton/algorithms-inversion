Counting Inversions Using Divide-and-Conquer

This page uses the merge sort method to count the number of inversions--pairs of numbers in which the earlier number is greater than the later one--in a list of numbers. This page imports the list from a local .txt file on the user's machine. In order to implement this, I've used the HTML5 FileReader API.

In order for this page to function correctly, the list of numbers must be a line-break (carriage return) delimited list in a text file.
